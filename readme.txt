Please follow the below instructions to run the assigned challenge.

1. Open the shared nodejs project in visual studio code IDE.

2. Open new integrated terminal by using menu Terminal -> New Terminal.

3. Run the below command to install node dependencies using npm 
    npm install

4. Run the below command to execute the code that takes number as input parameter
    node index.js --number 35853

5. Run the below command to run unit tests
    npm test
